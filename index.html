How It Works

1. User registers / logs in
2. Enters a domain name to scan
3. System runs security scan via OWASP ZAP
4. Raw results are analyzed and stored
5. If website passes → Generate certificate and allow download
6. If website fails → Show results with suggestions to fix
7. User can view scan history, download reports, and rescan site
